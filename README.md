## AI ER Diagram & Flowchart Generator

This project is a modern, dark-themed web app that generates ER diagrams or flowcharts from your app ideas or SQL using AI. Built with Next.js and Mermaid.js, it features a clean UI and instant diagram generation.

**Features:**
- Enter an app idea or SQL to generate a visual diagram
- Dark mode and modern design
- Powered by AI and Mermaid.js

## Getting Started

Run the development server:

```bash
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.
